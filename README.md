# 💳 UPI QR Generator

A simple and modern website to generate UPI payment QR codes instantly. Enter your UPI ID, name, amount, and note to create a QR code that works with all UPI apps.

## ✨ Features

* Instant QR Generation
* Live QR Preview
* Download QR
* Copy UPI Link
* Responsive Design
* Dark & Light Mode
